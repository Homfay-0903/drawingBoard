# 你画我猜 - 前端开发方案

## 一、项目概述

### 1.1 项目背景

将现有的单机绘画板应用重构为多人在线"你画我猜"游戏。

### 1.2 技术栈

| 技术             | 版本   | 用途         |
| ---------------- | ------ | ------------ |
| React            | 19.1.1 | UI框架       |
| TypeScript       | 5.8.3  | 类型安全     |
| Vite             | 7.1.6  | 构建工具     |
| React Router     | 7.13.1 | 路由管理     |
| Socket.io Client | ^4.7.2 | 实时通信     |
| Rough.js         | 4.6.6  | 手绘风格绘图 |

### 1.3 现有功能

- 绘图工具：矩形、箭头线、手绘线、文本
- 样式控制：颜色选择、画笔粗细
- 历史记录：撤销/重做
- 导出功能：导出为PNG图片

## 七、Socket通信封装

### 7.1 useSocket Hook

**文件**: `src/hooks/useSocket.ts`

```typescript
import { useEffect, useCallback } from "react";
import { io, Socket } from "socket.io-client";
import { useSocketContext } from "../context/socket-context";
import type { ClientEvents, ServerEvents } from "../types/socket";

export const useSocket = () => {
  const { socket, setSocket, setIsConnected, setPlayerId } = useSocketContext();

  const connect = useCallback(() => {
    const newSocket = io("http://localhost:3001", {
      transports: ["websocket"],
    });

    newSocket.on("connect", () => {
      setIsConnected(true);
      setPlayerId(newSocket.id || null);
    });

    newSocket.on("disconnect", () => {
      setIsConnected(false);
    });

    setSocket(newSocket);
  }, []);

  const disconnect = useCallback(() => {
    if (socket) {
      socket.disconnect();
      setSocket(null);
      setIsConnected(false);
    }
  }, [socket]);

  return { connect, disconnect, socket };
};
```

### 7.2 useGame Hook

**文件**: `src/hooks/useGame.ts`

```typescript
import { useCallback, useEffect } from "react";
import { useGameContext } from "../context/game-context";
import { useSocketContext } from "../context/socket-context";

export const useGame = () => {
  const { socket } = useSocketContext();
  const {
    setCurrentRoom,
    setPlayers,
    setGameStatus,
    setTimeLeft,
    setCurrentWord,
    setWordHints,
    addMessage,
    setIsDrawer,
  } = useGameContext();

  // 创建房间
  const createRoom = useCallback(
    (name: string, maxPlayers: number) => {
      socket?.emit("room:create", { name, maxPlayers });
    },
    [socket],
  );

  // 加入房间
  const joinRoom = useCallback(
    (roomId: string, nickname: string) => {
      socket?.emit("room:join", { roomId, nickname });
    },
    [socket],
  );

  // 监听房间更新
  useEffect(() => {
    if (!socket) return;

    socket.on("room:updated", ({ room }) => {
      setCurrentRoom(room);
      setPlayers(room.players);
    });

    socket.on("game:tick", ({ timeLeft }) => {
      setTimeLeft(timeLeft);
    });

    socket.on("chat:message", (data) => {
      addMessage({
        id: Date.now().toString(),
        ...data,
        timestamp: Date.now(),
      });
    });

    return () => {
      socket.off("room:updated");
      socket.off("game:tick");
      socket.off("chat:message");
    };
  }, [socket]);

  return { createRoom, joinRoom };
};
```

## 十、注意事项

1. **画布同步**: 使用增量同步策略，只同步新增的绘图元素
2. **状态管理**: 游戏状态由服务端控制，客户端只负责渲染
3. **错误处理**: 完善的网络断开重连机制
4. **性能优化**: 使用 React.memo 和 useMemo 优化组件渲染
5. **用户体验**: 添加加载状态和过渡动画
